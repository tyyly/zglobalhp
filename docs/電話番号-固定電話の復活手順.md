## 復活させたいとき（固定電話を再表示）

### 1. 復活ポイントを探す

各 HTML ファイルの復活ポイントには、すべて検索用マーカー **`固定電話復活`** を埋め込んであります。
プロジェクトルートで以下を実行すると、全21箇所が一覧で出ます。

```powershell
# PowerShell
Select-String -Path dev\site\*.html -Pattern "固定電話復活"
```

```bash
# Git Bash 等
grep -rn "固定電話復活" dev/site/*.html
```

### 2. コメントを解除する

復活ポイントは次の2パターンです。マーカー行の直後を見れば、どちらか判別できます。

**パターンA：固定電話の行をコメント解除するだけ**
（携帯はすでに別行で表示されているので、固定電話を「足す」だけ）

```html
<!-- 固定電話復活: 次行のコメント解除で固定電話を再表示 -->
<!-- <li>...固定電話...</li> -->   ← この行の <!-- と --> を外す
<li>...携帯...</li>
```

**パターンB：固定電話を戻し、携帯のみの行を消す**
（footer や 一覧表など、固定電話＋携帯を1行にまとめていた箇所）

```html
<!-- 固定電話復活: 下のコメント解除＋携帯のみの行を削除で元に戻る -->
<!-- <p>...固定電話 / 携帯...</p> -->   ← この行の <!-- と --> を外す（＝固定電話＋携帯の併記に戻る）
<p>...携帯のみ...</p>                    ← この行は削除する
```

> マーカー行（`<!-- 固定電話復活: ... -->`）自体は消しても残しても表示には影響しません。
> 残しておけば次回また切り替えるときに楽です。

### 3. JSON-LD（構造化データ）だけは手作業で戻す

`dev/site/index.html` の先頭付近にある JSON-LD は、JSON のためコメントが使えません。
値を直接書き換えてあります。固定電話に戻すときは次のように変更してください。

```jsonc
// 現在
"telephone": "+81-80-3999-0379",
// 固定電話に戻す場合
"telephone": "+81-476-29-5498",
```
