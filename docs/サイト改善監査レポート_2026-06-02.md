## 2. 画像・読込速度

- 方式: ファイル名・拡張子を変えない in-place 最適化（HTML/CSS参照を壊さない）
  - JPEG: 最大辺1920px超は縮小し quality80 progressive で再圧縮
  - `zglobal/` の車両・サービス写真: 実表示が600-700pxのため長辺1100pxへ縮小
  - PNG: 可逆最適化のみ
- 主な縮小実績: bg-features1.jpg 2.0MB→213KB / vehicle-alphard 514→250KB / vehicle-granace 329→129KB / service6-1 365→52KB / team・pricing系 各80-90%減

### 未実施（次フェーズ推奨）
- **WebP化 + `<picture>`フォールバック**: さらに30-50%減。とくに残る透過photo-PNG（man1.png 453KB, benefit3-1.png 382KB, dotted-map1.png 309KB, man2.png 215KB）はWebPが有効（透過保持のためJPEG化不可）
- **srcset によるレスポンシブ画像**（複数解像度）
- 最適化スクリプト: `dev/optimize_images.py`（引数: 対象ディレクトリ, 最大辺, JPEG品質）

## 3. SEO

### 実施済み
- 全5ページの `<title>` / `<meta description>` を事業キーワード（羽田・成田 / 空港送迎 / 役員送迎 / 観光チャーター / ゴルフ送迎 / 車種名）入りに刷新
- OGP（og:*）+ Twitterカード + self canonical + hreflang(ja/en/zh/ko) を全ページ付与
- robots.txt / sitemap.xml 追加

### 次フェーズ推奨
- 各ページに構造化データ追加（Service / FAQPage / AggregateRating schema）。index の LocalBusiness に geo座標を追加
- **title/description/OGP の多言語化**（本文はi18n.js で4言語化済みだがメタは日本語のみ→en/zh/ko版の検索最適化が偏る）
- Google ビジネスプロフィール整備（「成田市 ハイヤー」等の地域SEO）
- 内部リンク強化（サービス別アンカーへの自然な誘導）、装飾画像のalt補完

### 推奨キーワード（要点）
- 主要: 羽田送迎 / 成田送迎 / 空港送迎 / 高級ハイヤー / 役員送迎 / 観光チャーター / ゴルフ送迎
- 車両ロングテール: アルファード ハイヤー / グランエース Web会議 / ハイエース 大人数・ゴルフバッグ
